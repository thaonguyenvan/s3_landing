Use Cases

Lưu trữ và truyền tải nội dung Media & Assets

Cloud365 Storage cho phép bạn lưu trữ toàn bộ các file tài nguyên tĩnh: videos, nhạc, ảnh, Javascript, CSS,...Dịch vụ object storage của chúng tôi cho phép truy cập thông qua tất cả các công cụ hỗ trợ S3 API. Bạn có thể chia sẻ cho bất cứ ai quyền truy cập ở cả chế độ công khai và riêng tư.

Backups & Logs

Cloud365 Storage cung cấp môi trường bảo mật, ổn định và khả năng mở rộng không giới hạn, rất phù hợp để backup dữ liệu của bạn. Ngoài ra, lưu trữ log tập trung tại đây cũng giúp bạn không còn phải quan tâm về vấn đề xoay vòng dữ liệu.

Host static web

Cloud365 Storage cho phép bạn upload các ứng dụng website tĩnh (chỉ bao gồm HTML, CSS, JS..., các file ảnh, video). Website sẽ được hiển thị dựa theo domain được cung cấp sẵn.

Tích hợp với các công cụ

Cloud365 Storage có thể tích hợp với tất cả các công cụ tương thích với S3 API.

Cyberduck

Công cụ mã nguồn mở cho Windows và macOS với giao diện đồ họa phục vụ việc quản lý file.

FileZilla

Công cụ transfer file hỗ trợ khác nhiều giao thức (FTP, SFTP, ...). S3 chỉ được hỗ trợ với bản PRO.

Transmit

Công cụ dành cho macOS với giao diện đồ họa phục hỗ trợ khá nhiều loại server lưu trữ.

S3 Browser

Công cụ được phát triển dành cho Amazon S3, cung cấp giao diện để quản lý file.
